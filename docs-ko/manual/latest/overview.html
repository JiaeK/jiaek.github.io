블록(Blocks)

{}으로 구분된 블록의 마지막 표현식은 암시적으로 반환됩니다(함수 본문 포함). JavaScript에서는 함수 본체(body)의 고유한 로컬 범위를 가지고 있기 때문에, 즉시 호출되는 함수 표현식을 통해서만 시뮬레이션해볼 수 있습니다.

JavaScriptReScript
let result = (function() {
  const x = 23;
  const y = 34;
  return x + y;
})();
let result = {
  let x = 23
  let y = 34
  x + y
}
